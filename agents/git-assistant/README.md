# 🎋 AI Git Commit Assistant

Generate high-quality commit messages directly from your code changes. No more `git commit -m "fixed stuff"`.

## 🚀 Features
- **Diff Analysis**: Understands logic changes across multiple files.
- **Conventional Commits**: Supports `feat:`, `fix:`, `refactor:`, etc.
- **Choice of Options**: Provides 3 variations from concise to detailed.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
