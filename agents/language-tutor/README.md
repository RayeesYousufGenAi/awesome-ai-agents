# 🌎 AI Language Tutor

A conversational AI partner that helps you practice speaking new languages with real-time feedback and corrections.

## 🚀 Features
- **Multi-Language Support**: Practice Spanish, French, Japanese, and more.
- **Active Corrections**: Points out grammar mistakes and suggests natural alternatives.
- **Natural Conversation**: Keeps you engaged with relevant follow-up questions.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit (Chat Interface)

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
