# 📅 AI Video Content Calendar

Never run out of ideas. This agent generates a full 30-day social media strategy tailored to your niche and platforms.

## 🚀 Features
- **Niche Focus**: Ideas optimized for Finance, Tech, Lifestyle, etc.
- **Platform-Specific Hooks**: Different strategies for TikTok vs Long-form YouTube.
- **Viral Forecasting**: Suggests trending formats and weekly themes.

## 🛠️ Tech Stack
- LangChain / OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
