# 🌐 AI Web Researcher

Enter any topic or URL — get AI-curated research summaries, key facts, trends, and actionable insights.

## ✨ Features
- 🔍 Deep topic research with GPT-4
- 🌐 URL scraping and content analysis
- 📋 Structured summaries with key takeaways
- ❓ Ask specific questions about any webpage

## 🛠️ Tech Stack
- **OpenAI GPT-4** — Research and analysis
- **BeautifulSoup** — Web scraping
- **Streamlit** — Tabbed UI

## 🚀 Quick Start

```bash
cd agents/web-researcher
pip install -r requirements.txt
export OPENAI_API_KEY="your-key"
streamlit run app.py
```
