# 📝 Meeting Minutes Summarizer

Turn messy meeting transcripts into professional executive summaries and clear action items.

## 🚀 Features
- **Action Item Extraction**: Identifies responsibilities and deadlines for team members.
- **Multi-Level Summaries**: Choose between Concise, Detailed, or Executive outputs.
- **Decision Tracking**: Highlights finalized outcomes of the meeting.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
