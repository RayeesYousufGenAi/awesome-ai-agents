# 📊 AI Data Analyst

Upload any CSV and get instant AI-powered data analysis — summaries, trends, outliers, and business insights.

## ✨ Features
- 📁 Upload and preview CSV data
- 🧠 GPT-4 powered data analysis
- 📈 Auto-generated visualizations
- 💡 Business insights and recommendations
- ❓ Ask custom questions about your data

## 🛠️ Tech Stack
- **OpenAI GPT-4** — Data analysis
- **Pandas** — Data processing
- **Streamlit** — Charts and UI

## 🚀 Quick Start

```bash
cd agents/data-analyst
pip install -r requirements.txt
export OPENAI_API_KEY="your-key"
streamlit run app.py
```
