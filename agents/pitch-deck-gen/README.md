# 🚀 AI Startup Pitch Deck Generator

Secure the bag. This agent uses VC-standard frameworks to build a compelling 10-slide narrative for your startup.

## 🚀 Features
- **Narrative Logic**: Structures your story from Problem to The Ask.
- **Market Sizing**: Includes frameworks for TAM, SAM, and SOM.
- **VC-Style Feedback**: Suggests what to emphasize to attract investors.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
