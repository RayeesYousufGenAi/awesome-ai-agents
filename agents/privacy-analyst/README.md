# 🔐 AI Privacy Policy Analyst

Don't let companies hide their data tracking in fine print. This agent audits privacy policies and gives a clear "Privacy Grade".

## 🚀 Features
- **Intrusiveness Audit**: Identifies exactly what personal behavior is being tracked.
- **Selling Detection**: Explicitly flags if data is shared with third-party advertisers.
- **Letter Grading**: Easy-to-understand A-F ranking for privacy safety.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
