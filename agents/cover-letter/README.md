# 💌 AI High-Conversion Cover Letter

Stop sending generic templates. This agent synthesizes your unique background with specific job requirements for a winning letter.

## 🚀 Features
- **Impact-First Matching**: Connects your past wins to their future needs.
- **Professional Tone**: Balanced, assertive, and non-robotic.
- **Brevity Optimization**: Keeps it short enough for busy recruiters to actually read.

## 🛠️ Tech Stack
- LangChain / OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
