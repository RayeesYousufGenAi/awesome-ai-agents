# 📞 AI Cold Call Script Writer

Turn dreaded outbound calls into high-conversion conversations with scripts that bypass the "Spam" filter in people's brains.

## 🚀 Features
- **Pattern Interruption**: Scripts designed to break the standard sales-talk mold.
- **Objection Combat**: Pre-written rebuttals for the 5 most common cold-call rejections.
- **Discovery Focus**: Asks the right questions to qualify leads fast.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
