# 📧 AI Email Writer

An AI-powered agent that generates professional emails based on user input and tone.

## 🚀 Features
- Generate complete emails with subject line
- Supports multiple tones (Formal, Casual, Friendly)
- Clean and simple UI using Streamlit

## 🛠 Tech Stack
- Streamlit
- Google Gemini API
