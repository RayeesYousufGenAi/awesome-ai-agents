# 🗄️ AI SQL Query Optimizer

Speed up your apps. This agent audits your database queries to find slowness and suggests production-ready optimizations.

## 🚀 Features
- **Bottleneck Detection**: Finds full table scans and inefficient joins.
- **Index Suggestions**: Recommends exactly which columns to index for speed.
- **Multi-DB Support**: Optimized for PostgreSQL, MySQL, and SQLite.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
