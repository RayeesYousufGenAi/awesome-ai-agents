# 🎵 AI Lyrics & Songwriter

Unleash your musical creativity with an AI agent that writes full lyrics across various genres and moods.

## 🚀 Features
- **Genre-Specific**: Tailors rhyme schemes and vocabulary for Rap, Pop, Rock, etc.
- **Storytelling**: Converts your basic idea into a full 3-minute song structure.
- **Stage Directions**: Includes musical cues like "[Bass drops]" or "[Guitar solo]".

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
