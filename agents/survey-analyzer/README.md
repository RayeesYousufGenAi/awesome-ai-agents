# 📊 AI Survey & Feedback Analyzer

Transform hundreds of raw customer reviews or internal surveys into a clear, actionable dashboard of insights.

## 🚀 Features
- **Sentiment Mapping**: Instantly gauges the emotional tone of your audience.
- **Thematic Clustering**: Groups feedback by topic (e.g., Pricing, UX, Support).
- **Prioritization Engine**: Highlights the most critical issues to fix first.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
