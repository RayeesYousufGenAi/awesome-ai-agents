# 📈 SEO Meta-Content Optimizer

An AI agent that analyzes your content and generates SEO-optimized metadata to help you rank higher on search engines.

## 🚀 Features
- **Smart Titles**: Generates catchy, CTR-focused headlines.
- **Meta Descriptions**: Concise summaries optimized for Google.
- **Keyword Extraction**: Identifies LSI keywords to prevent keyword stuffing.
- **Readability Tips**: Actionable feedback to improve content quality.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
