# 🎬 Niche Movie & Book Recommender

A cultural discovery agent that skips generic bestseller lists and finds the perfect media match for your current "vibe".

## 🚀 Features
- **Vibe-Based Curation**: Uses semantic understanding to match moods (e.g., "cyberpunk nostalgia").
- **Obscurity Bias**: Prioritizes high-quality hidden gems over mainstream blockbusters.
- **Cross-Media**: Get both book and movie matches for a unified experience.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
