# 💻 AI Tech Stack Recommender

Avoid tech debt. Get a CTO-level architectural recommendation for your next project based on scale and complexity goals.

## 🚀 Features
- **Tailored Selections**: Picks NoSQL vs SQL based on your data needs.
- **Modern Standards**: Recommends Next.js, Vite, FastAPI, etc.
- **Scalability Focus**: Considers your hosting and CI/CD needs from Day 1.

## 🛠️ Tech Stack
- LangChain / OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
