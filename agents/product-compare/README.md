# ⚖️ AI Product Comparison Helper

Make informed buying decisions. side-by-side spec comparisons with a clear verdict based on your personal use case.

## 🚀 Features
- **Semantic Comparison**: Goes beyond just specs to compare real-world value.
- **Verdict Engine**: Gives a direct recommendation (Buy A or Buy B).
- **Use-Case Prioritization**: Tailors advice for photographers, gamers, or business pros.

## 🛠️ Tech Stack
- LangChain / OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
