# 📧 Cold Email Personalizer

Generate hyper-personalized outreach emails that get replies by analyzing prospect data and company bios.

## 🚀 Features
- **Deep Hook Extraction**: Finds unique points in bios to use as email openers.
- **Tone Control**: Choose from Professional, Witty, or Direct.
- **Lead Gen Ready**: Optimized for conversion and low-friction CTAs.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
