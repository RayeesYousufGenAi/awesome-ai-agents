# 🌍 AI Subtitle Translator

Translate video subtitles (SRT) into multiple languages while preserving timestamps and context.

## 🚀 Features
- **SRT Format Support**: Maintains sequence numbers and timecodes.
- **Contextual Accuracy**: Translates with the flow of the conversation, not just line-by-line.
- **Timing Optimization**: Ensures translated text isn't excessively long for the screen.

## 🛠️ Tech Stack
- LangChain / OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
