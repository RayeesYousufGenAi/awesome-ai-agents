# 👨‍🏫 AI Coding Interview Coach

Prepare for your next big tech role with an AI-powered interviewer that conducts mock technical rounds and provides instant feedback.

## 🚀 Features
- **Role-Specific Questions**: Tailored for Frontend, Backend, Data, and Fullstack roles.
- **Difficulty Levels**: Scales from Junior to Senior level concepts.
- **Interactive Feedback**: Evaluates your code and logic in real-time.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit (Chat UI)

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
