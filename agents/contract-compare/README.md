# 📜 AI Contract Comparison Bot

Identify hidden risks in contract revisions. This agent "diffs" legal documents for meaning, not just text.

## 🚀 Features
- **Semantic Diffing**: Finds changes in obligation even if the wording is slightly different.
- **Risk Assessment**: Flags if a new version is significantly worse for you.
- **Side-by-Side Analysis**: Clear breakdown of added or removed clauses.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
