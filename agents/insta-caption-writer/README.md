# 📸 AI Instagram Caption Writer

Generate engagement-focused Instagram captions with hooks, emojis, and sets of trending hashtags.

## 🚀 Features
- **Engagement Hooks**: Captions designed to stop the scroll.
- **Mood Tones**: Switch between Savage, Inspirational, and Fun.
- **Hashtag Bundles**: Automatically includes relevant hashtags for visibility.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
