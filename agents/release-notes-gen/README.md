# 🆕 AI Release Notes Generator

Bridge the gap between engineering and marketing. Convert raw commit logs and tickets into beautiful product updates.

## 🚀 Features
- **User-Centric Language**: Decodes technical jargon into benefits.
- **Automatic Categorization**: Groups changes into Features, Fixes, and Speed.
- **Engagement-Focused**: Written in a style that excites your users about the progress.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
