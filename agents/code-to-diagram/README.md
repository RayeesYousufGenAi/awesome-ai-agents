# 🏗️ AI Code-to-Architecture Diagram

Visualize complex systems instantly. This agent parses codebases or text descriptions and outputs Mermaid.js diagram code.

## 🚀 Features
- **Automatic Mapping**: Identifies relationships between Frontend, Backend, and DB.
- **Mermaid Support**: Generates valid code for flowcharts, sequence, and class diagrams.
- **Architecture Visualization**: Helps developers explain their systems during PRs.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
