# 🧪 AI Unit Test Generator

Automate your testing workflow by generating high-coverage test suites for your Python functions and classes.

## 🚀 Features
- **Pytest/Unittest Support**: Choose your preferred framework.
- **Edge Case Detection**: Automatically tests for nulls, empty inputs, and boundary conditions.
- **Mocking Integration**: Suggests mocks for API calls or database connections.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
