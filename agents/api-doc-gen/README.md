# 📖 AI API Documentation Bot

Convert your raw backend code into human-readable, professional Markdown documentation in seconds.

## 🚀 Features
- **Multi-Language**: Works with FastAPI, Flask, Express.js, Spring Boot, and more.
- **OpenAPI Ready**: Structures data for easy copy-pasting into technical wikis.
- **Example Generation**: Automatically builds sample JSON requests and responses.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
