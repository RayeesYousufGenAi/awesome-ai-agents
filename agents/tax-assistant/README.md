# 💸 AI Tax Prep Assistant

Navigate the complexity of taxes. This agent helps freelancers and small business owners identify eligible deductions.

## 🚀 Features
- **Region-Specific**: Tailors advice for US, India, UK, and Canada.
- **Deduction Finder**: Identifies write-offs you might have missed.
- **Mistake Prevention**: Highlights common red flags that trigger audits.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
