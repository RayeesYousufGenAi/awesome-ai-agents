# 📄 AI Resume Builder

Create professional resumes from user input with AI-powered content generation and PDF export.

## ✨ Features

- 📝 Form inputs for personal info (name, email, phone)
- 💼 Multiple work experience entries
- 🎓 Multiple education entries
- 🏷️ Skills list input
- 🧠 GPT-4 powered professional content generation
- 📄 PDF export with 3 template options (Modern, Classic, Minimal)
- ⬇️ Direct download button for generated PDF

## 🛠️ Tech Stack

- **OpenAI GPT-4** — Resume content generation
- **FPDF2** — PDF export and formatting
- **Streamlit** — Web UI and forms
- **python-dotenv** — Environment configuration

## 🚀 Quick Start

```bash
cd agents/resume-builder
pip install -r requirements.txt
export OPENAI_API_KEY="your-key"
streamlit run app.py
```
