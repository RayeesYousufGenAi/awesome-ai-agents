# 👤 Social Media Bio Optimizer

Your bio is your first impression. This agent rewrites your profile for maximum professional impact and conversion.

## 🚀 Features
- **Platform Specific**: Adjusts tone and length for LinkedIn, X, and Instagram.
- **Proof-Focus**: Emphasizes your achievements and "Proof of Work".
- **CTA Injection**: Adds clear Calls to Action to drive followers or clicks.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
