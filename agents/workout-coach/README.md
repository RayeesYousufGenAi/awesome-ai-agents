# 💪 AI Personal Workout Planner

Stop guessing in the gym. Get a data-driven training routine designed specifically for your goals and available gear.

## 🚀 Features
- **Custom Splits**: High-intelligence volume and frequency management.
- **Equipment Injection**: Works whether you have a full gym or just bodyweight.
- **Level Scaling**: Adjusts intensity and complexity for beginners to athletes.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
