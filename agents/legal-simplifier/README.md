# ⚖️ Legal jargon Simplifier

Decode complex legal contracts and Terms of Service into simple, understandable English for non-lawyers.

## 🚀 Features
- **Legalese Decoder**: Converts jargon like "Indemnification" and "Arbitration" into plain English.
- **Red Flag Detector**: Highlights risky clauses you might want to know before signing.
- **Summary Mode**: Quick breakdown of rights and obligations.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
