# 🔗 LinkedIn Post Generator

An AI agent that creates high-engagement LinkedIn posts from topics, articles, or links. It supports multiple writing styles and target audiences.

## ✨ Features
- **4 Writing Styles**: Viral, Professional, Story-based, and Casual.
- **Hook Optimization**: Focuses on the first 2 lines to stop the scroll.
- **Audience Targeting**: Tailors tone for specific professional groups.
- **Hashtag Generator**: Suggests relevant tags for visibility.

## 🛠️ Tech Stack
- **Framework**: Streamlit
- **AI**: LangChain + OpenAI GPT-4

## 🚀 Quick Start

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the app:**
   ```bash
   streamlit run app.py
   ```

3. **Usage:**
   - Enter your OpenAI API Key in the sidebar.
   - Paste your topic or URL.
   - Choose a style and click **Generate**.
