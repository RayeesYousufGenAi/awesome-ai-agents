# 🌤️ AI Daily Standup Generator

Save time on reporting. Turn messy notes and browser history into professional, structured status updates for your team.

## 🚀 Features
- **Yesterday/Today/Blockers**: The standard Agile format, perfectly structured.
- **Narrative Cleanup**: Converts "fixed that bug in the thing" into "Resolved critical UI rendering issue".
- **Slack-Ready**: Optimized for quick copy-pasting.

## 🛠️ Tech Stack
- LangChain / OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
