# 📇 PDF Flashcard Creator

A study agent that parses academic PDFs and generates high-impact flashcards for active recall.

## 🚀 Features
- **PDF Parsing**: Automatically reads and understands complex study materials.
- **Active Recall**: Generates question-answer pairs optimized for memorization.
- **Anki-Ready**: Output can be easily copy-pasted into Anki or Quizlet.

## 🛠️ Tech Stack
- `pypdf`
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
