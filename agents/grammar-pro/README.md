# ✍️ AI Grammar & Tone Optimizer

A professional editing agent that goes beyond spellcheck to refine your flow, clarity, and emotional impact.

## 🚀 Features
- **Tone Shifting**: Instantly switch between "Witty" and "Academic".
- **Clarity Audit**: Identifies and removes wordy or confusing phrases.
- **Impact Scoring**: Explains why the new version works better for your target audience.

## 🛠️ Tech Stack
- LangChain / OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
