# 💸 AI Expense Categorizer

Stop manual bookkeeping.Paste your raw bank data and let AI categorize your spending and find saving opportunities.

## 🚀 Features
- **Auto-Categorization**: Sorts transactions into Food, Travel, Rent, etc.
- **Budget Insights**: Provides a summary of total spending by category.
- **Saving Tips**: AI notes patterns and suggests where to cut back.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
