# 🎨 AI Art Prompt Engineer

Convert simple ideas into professional-grade generative art prompts optimized for Midjourney, DALL-E 3, and Stable Diffusion.

## 🚀 Features
- **Engine Optimization**: Tailors keyword weights and parameters (like `--ar` for Midjourney).
- **Style Injection**: Suggests lighting, textures, and artistic movements.
- **Negative Prompts**: Automatically generates what to exclude for cleaner results.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
