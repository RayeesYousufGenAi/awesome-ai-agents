# ✈️ AI Travel Itinerary Planner

Your personal travel agent that crafts customized multi-day itineraries based on your budget and interests.

## 🚀 Features
- **Destination Mapping**: In-depth knowledge of global tourist and local spots.
- **Budget-Conscious**: Tailors suggestions for Shoestring, Moderate, or Luxury tiers.
- **Pro-Tips**: Includes local advice (e.g., transport, scams to avoid, best times to visit).

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
