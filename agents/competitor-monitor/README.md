# 📡 Competitive Intelligence Bot

Stay offensive. This agent analyzes your competition's marketing and product moves to suggest winning counter-strategies.

## 🚀 Features
- **Strategic Auditing**: Breaks down competitor value propositions.
- **Moat Analysis**: Identifies what makes their business defensible (or vulnerable).
- **Tactical Counter-Moves**: Suggests specific pivot points for your business.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
