# 🪴 AI House Plant Doctor

Save your greens. Identify plant diseases and care issues instantly with an expert botanical knowledge base.

## 🚀 Features
- **Symptom Analysis**: Matches yellowing, drooping, or spots to specific issues.
- **Recovery Calendars**: Provides a structured 2-week plan to nurse plants back to health.
- **Care Optimization**: Unique tips for Monstera, Pothos, Succulents, and more.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
