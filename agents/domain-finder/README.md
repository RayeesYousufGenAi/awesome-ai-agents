# 🌎 AI Domain & Startup Namer

Stop wasting time on domain search bars. This agent brainstorms short, catchy, and available-feeling domains for your next project.

## 🚀 Features
- **Extensive Brainstorming**: Leverages LLM creativity to find unique portmanteaus.
- **TLD Optimization**: Adjusts suggestions based on whether you want a .com, .ai, or .io.
- **Tech Brand Scoring**: Analyzes how "premium" a name feels for the tech industry.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
