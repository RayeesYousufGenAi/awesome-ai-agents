# 📜 AI Patent Searcher

Evaluate the novelty of your inventions and business ideas with an AI trained on patent structures and prior art analysis.

## 🚀 Features
- **Novelty Scoring**: Instant feedback on how unique your idea is.
- **Prior Art Identification**: Suggests existing technologies that might conflict with your claim.
- **Filing Guidance**: Legal-style advice on what to emphasize for a successful patent.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
