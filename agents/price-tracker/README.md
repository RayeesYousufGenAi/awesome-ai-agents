# 🏷️ Smart Price Alert Analyst

A shopping assistant that helps you decide if a product is at its best price or if you're being overcharged.

## 🚀 Features
- **Deal Evaluation**: Provides a Buy/Wait/Run verdict based on price points.
- **Seasonal Intelligence**: Knows when major sales are likely to drop.
- **Lifecycle Analysis**: Considers if a newer model is about to be released.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
