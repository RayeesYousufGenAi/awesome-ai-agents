# 🔍 AI Code Reviewer

Paste any code and get instant AI-powered review — detects bugs, suggests optimizations, flags security issues, and recommends best practices.

## ✨ Features
- 🐛 Bug detection and fixes
- ⚡ Performance optimization tips
- 🔒 Security vulnerability scanning
- 📊 Code quality score (1-10)
- 🌐 Supports Python, JavaScript, Java, Go, Rust, and more

## 🛠️ Tech Stack
- **OpenAI GPT-4** — Code analysis
- **Streamlit** — Web UI

## 🚀 Quick Start

```bash
cd agents/code-reviewer
pip install -r requirements.txt
export OPENAI_API_KEY="your-key"
streamlit run app.py
```
