# 💼 AI Job Post Generator

Attract top talent. Create professional, inclusive, and exciting job descriptions that stand out on LinkedIn and Indeed.

## 🚀 Features
- **Tone Matching**: From "Serious Enterprise" to "Disruptive Startup".
- **Requirements Synthesis**: Reorganizes messy bullet points into structured roles.
- **SEO Optimized**: Written to rank well in job search algorithms.

## 🛠️ Tech Stack
- LangChain / OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
