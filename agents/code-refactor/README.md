# 🛠️ AI Code Refactorer

Turn "spaghetti" code into clean, maintainable architecture. Focus on readability, performance, or security with one click.

## 🚀 Features
- **Rule-Based Refactoring**: Prioritize DRY, Readability, or Performance.
- **Variable Renaming**: Automatically suggests more descriptive names.
- **Security Hardening**: Identifies and fixes basic code-level vulnerabilities.

## 🛠️ Tech Stack
- LangChain / OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
