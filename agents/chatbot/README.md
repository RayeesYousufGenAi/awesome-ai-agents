# 🤖 Smart Chatbot Agent

An AI-powered conversational assistant with conversation memory using LangChain and GPT-4.

## ✨ Features
- 💬 Natural conversation with GPT-4
- 🧠 Remembers last 15 messages for context
- 🎨 Beautiful Streamlit chat interface
- ⚡ Streaming responses

## 🛠️ Tech Stack
- **LangChain** — Conversation chain with memory
- **OpenAI GPT-4** — Language model
- **Streamlit** — Chat UI

## 🚀 Quick Start

```bash
cd agents/chatbot
pip install -r requirements.txt
export OPENAI_API_KEY="your-key"
streamlit run app.py
```
