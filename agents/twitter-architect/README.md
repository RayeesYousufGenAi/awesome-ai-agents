# 🐦 Twitter/X Thread Architect

Repurpose your long-form content into viral, engaging Twitter threads with the perfect hook and structure.

## 🚀 Features
- **Hook Optimization**: Analyzes your content for the most scroll-stopping opener.
- **Narrative Logic**: Ensures a logical flow across the entire thread.
- **Style Presets**: Choose between Educational, Controversial, or Storytelling styles.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
