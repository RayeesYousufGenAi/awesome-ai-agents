# 🍳 AI Recipe Generator

The ultimate kitchen companion. Generate gourmet recipes based exactly on the ingredients you have available.

## 🚀 Features
- **Fridge-to-Plate**: Minimizes food waste by utilizing your current inventory.
- **Dietary Aware**: Filters for Vegan, Keto, Gluten-Free, and more.
- **Chef's Secrets**: Includes professional tips to elevate home-cooked meals.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
