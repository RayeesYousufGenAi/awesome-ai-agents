# 🏷️ AI Brand Name Generator

Generate memorable and industry-specific brand names with psychological justifications and taglines.

## 🚀 Features
- **Industry Selection**: Tailors names for Tech, Fashion, Food, and more.
- **Vibe Control**: Choose from Corporate to Playful tones.
- **Branding Logic**: Explains the psychology behind each name suggestion.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
