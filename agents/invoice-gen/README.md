# 🧾 AI Self-Employed Invoice Generator

Automate your billing. This agent takes messy work logs and transforms them into professional, itemized invoice tables.

## 🚀 Features
- **Log Parsing**: Automatically converts "2.5 hours on UI" into billing line items.
- **Auto-Calculation**: Handles totals, taxes, and hourly rate multiplying.
- **Professional Formatting**: Optimized for quick copy-pasting into PDF editors.

## 🛠️ Tech Stack
- LangChain
- OpenAI GPT-4o
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
