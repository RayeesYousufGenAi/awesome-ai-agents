# 📈 Stock Market Sentiment Analyst

An AI agent that combines real-world market news (`yfinance`) with LLM analysis to determine market sentiment for any stock ticker.

## 🚀 Features
- **Ticker Support**: Works with any symbol supported by Yahoo Finance.
- **News Integration**: Pulls latest headlines and analyzes their impact.
- **Sentiment Scoring**: 1-10 score with detailed bull/bear drivers.

## 🛠️ Tech Stack
- `yfinance`
- LangChain
- OpenAI GPT-4
- Streamlit

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
```
